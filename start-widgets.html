Scriptable Widgets

Copy any script into the Scriptable app on iOS. Each script is pre-configured with your keys. Add a widget to your home screen and select the matching script.

Setup: Install Scriptable (free on App Store) → tap + → paste the copied script → name it → long-press home screen → add Scriptable widget → select Medium size → choose the script.
Widgets needing Stripe or Telnyx keys: replace the placeholder at the top of the script with your actual key.
